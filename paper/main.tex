\section{问题重述}

\subsection{问题背景与优化目标}
本项目考察的是一个带有资源调度与路径规划性质的沙漠生存游戏。玩家需要凭借一张地图，从起点出发，利用初始资金购买特定数量的水和食物，途经不同的地形区域（如普通节点、矿山、村庄），最终在规定的截止日期或之前抵达终点。本题的优化目标是：在确保玩家成功存活并按时到达终点的前提下，通过合理规划行进路线与物资调度，使得游戏结束时\textbf{保留尽可能多的资金}。

\subsection{核心规则与约束条件}
玩家在游戏中的决策受到以下多重物理与规则约束：
\begin{itemize}
    \item \textbf{负重与生存约束：} 每天玩家拥有的水和食物质量之和不能超过负重上限。若未到达终点而水或食物已耗尽，视为游戏失败。
    \item \textbf{天气与环境影响：} 每天的天气为“晴朗”、“高温”、“沙暴”三种状况之一。沙暴日玩家必须在原地停留。
    \item \textbf{节点交互系统：} 第0天可在起点处用初始资金以基准价格购买水和食物，但不能多次购买；在村庄停留时可用剩余资金随时购买水和食物，每箱价格为基准价格的2倍；在矿山停留可通过挖矿获得资金，但到达矿山当天不能挖矿；到达终点后可退回剩余的水和食物，每箱退回价格为基准价格的一半。
    \item \textbf{消耗倍率：} 玩家在原地停留一天消耗的资源数量为基础消耗量，行走一天消耗的资源数量为基础消耗量的2倍，挖矿消耗的资源数量为基础消耗量的3倍。
\end{itemize}

\subsection{问题一的具体任务}
问题一设定为\textbf{假设只有一名玩家}，且属于全局确定性环境（即\textbf{在整个游戏时段内每天天气状况事先全部已知}）。任务要求建立数学模型，给出一般情况下玩家的最优策略，并利用该模型求解附件中的“第一关”和“第二关”。
\section{问题分析}

\subsection{问题一的分析}
问题一要求在全局天气信息完全已知的情况下，为单名玩家制定使得最终保留资金最大化的行动策略。针对该问题，我们从以下四个维度展开分析：

\subsubsection{问题特征分析}
本问题是一个典型的\textbf{确定性多阶段离散状态转移问题}。在“第一关”与“第二关”的设定中，玩家在整个游戏周期（30天）内面临的环境参数（每日天气序列、地图节点拓扑、基础物价等）完全已知且固定，不存在随机变量或概率博弈。因此，该问题本质上是在带权有向图的时间扩展网络上寻找满足约束条件的最优路径。

\subsubsection{关键变量界定}
为了精确描述玩家的状态转移，需要追踪的核心变量包括：
\begin{itemize}
    \item \textbf{时空变量：} 当前所处天数 $t$ 以及玩家当天所在的地图节点 $i$。
    \item \textbf{决策变量：} 玩家每日采取的具体动作（停留、行走或挖矿），以及在特定节点（起点或村庄）的物资购买量。
    \item \textbf{状态变量：} 每日结算后的剩余资金量 $M_t$、水量 $W_t$ 与食物量 $F_t$。
\end{itemize}

\subsubsection{模型难点剖析}
本题的核心难点在于\textbf{多重物理约束的强耦合性}与\textbf{非线性逻辑}的转化：
\begin{itemize}
    \item \textbf{负重与消耗的博弈：} 负重上限（1200 kg）严格限制了玩家的单次续航能力，而不同天气（晴朗、高温、沙暴）和不同动作（消耗倍率1倍至3倍不等）会导致资源消耗呈动态变化。玩家必须在“携带重物缓慢行进”与“轻装上阵依赖沿途高价补给”之间寻找平衡点。
    \item \textbf{非线性逻辑条件：} “沙暴日强制停留”、“仅能在村庄随时购买且价格翻倍”、“到达矿山首日不可挖矿”等游戏规则构成复杂条件约束。若直接采用传统公式表达，易导致模型非线性化并增加求解难度。
\end{itemize}

\subsubsection{可行求解思路}
鉴于时间跨度（30天）与空间节点（关卡一27个，关卡二64个）的规模均在有限可控范围内，且全局信息确定，我们排除启发式算法，采用\textbf{混合整数线性规划（MILP）模型}进行全局精确求解。
具体而言，引入0-1整数变量刻画玩家每日的位置与离散动作，并利用\textbf{大 M 法（Big-M Method）}将村庄购买等条件判断（If-Else逻辑）转化为线性不等式约束。最后，以游戏终局资金与折现物资价值总和最大化为目标函数，建立全局规划模型，确保所得策略具有理论最优性。
\subsection{问题二的分析}

问题二在问题一的基础上引入了核心的环境随机性变量：玩家仅知晓当天的天气状况，而对未来的天气演变未知（仅掌握天气的先验概率分布）。这一条件的改变，使得原本的确定性路径规划问题发生了本质的跃迁。

\begin{itemize}
    \item \textbf{决策范式的转变：} 由于未来天气的随机性，玩家无法再像问题一那样在第0天就规划出一条固定不变的“绝对最优路线”。任何静态的路径在遭遇连续极端恶劣天气（如连续高温或沙暴）时，都极易导致物资耗尽而游戏失败。因此，本题的求解目标从“寻找一条最优路径”转变为“寻找一个最优策略（Optimal Policy）”——即在任意给定日期、任意位置、任意物资储备与当天天气状态下，指导玩家做出当前期望收益最大的动作选择。
    \item \textbf{模型的映射与降维：} 这种带有随机状态转移与序贯决策特征的寻优过程，完美契合了\textbf{马尔可夫决策过程（Markov Decision Process, MDP）}的理论框架。为求解最大期望保留资金，我们需要构建高维状态张量，并利用动态规划中的逆向归纳法（Backward Induction），依据贝尔曼最优方程（Bellman Optimality Equation）从最后一天向第0天进行值迭代。此外，针对第四关村庄可随时购买任意数量物资的设定，若直接遍历购买数量将导致计算复杂度呈平方级爆炸，需利用状态价值的连续性与逆向传播机制进行降维优化。
\end{itemize}

\subsection{问题三的分析}

问题三将游戏场景从“单人对抗自然环境”升级为“多玩家零和/非零和博弈”。题干中引入了极其严苛的资源竞争惩罚机制：若多名玩家同日行走同路，资源消耗将飙升至基础消耗的 $2k$ 倍；同日同村购买，物价翻倍；同日同矿开采，基础收益平摊。这使得玩家的决策不仅受限于自然环境，更受到其他竞争者策略的强力制约。

\begin{itemize}
    \item \textbf{第五关分析（完全信息静态博弈）：} 第五关天气事先已知，且行动方案不可更改。由于去往矿山的必经之路存在极高的踩踏风险，该问题本质上演变为一个经典的\textbf{拥挤博弈（Congestion Game）}。传统的空间寻路算法在此失效，博弈的核心坍缩为“时间错峰调度”与“利益抢夺”。部分常规思路倾向于通过错峰出行避开惩罚，但经过深度的边际效益分析可知，避险所节约的物资成本可能远无法弥补出让核心矿区开采权所带来的高昂机会成本。因此，我们需要构建一个涵盖所有合理时间调度与绕路方案的巨型策略池，通过严密的逐日碰撞物理模拟器计算出多方的真实资金支付矩阵（Payoff Matrix），并利用单纯形法求解出玩家的\textbf{混合策略纳什均衡（Mixed Strategy Nash Equilibrium）}，以探究在该类“囚徒困境”下的最优解。
    
    \item \textbf{第六关分析（不完全信息动态博弈）：} 第六关结合了问题二的环境随机性与问题三的竞争博弈性，且玩家仅能获取周边视野（局部状态），每天需要进行动态交互决策。这构成了极为复杂的\textbf{随机马尔可夫博弈（Stochastic Markov Games）}。由于未来的天气与敌方动向均不可测，玩家必须在每个决策日综合各天气发生的概率，计算次日的期望收益与拥挤惩罚期望。我们将通过每日滚动构建“期望支付矩阵（Expected Payoff Matrix）”，并结合线性规划进行逐日纳什均衡求解，从而实现规避风险与利润最大化的动态平衡。
\end{itemize}
\section{模型假设}
鉴于题目中已经给出了较为详尽的物理约束与游戏规则，为了确保数学模型的严谨性与求解的可行性，我们对实际游戏过程中的部分隐性条件进行如下合理假设：

\begin{enumerate}
    \item \textbf{假设玩家具有完全理性：} 玩家在整个游戏过程中具有全局最优视野，始终以最终保留资金最大化为唯一目标进行决策，不存在操作失误或非理性的随意探索行为。
    \item \textbf{假设物资交易行为瞬间完成：} 玩家在起点或村庄购买水和食物时 [cite: 11, 13]，交易过程不额外消耗当天的行动时间。同时假设村庄的物资储备无限大，不受其他因素影响。
    \item \textbf{假设资源在储存与携带过程中无额外非线性损耗：} 水和食物除了按照每日天气和行动状态的设定被消耗外 [cite: 10]，不会因天气炎热、沙暴环境或存储时间过长而发生自然蒸发、变质或意外丢失。
    \item \textbf{假设行动状态具有时间离散性与动作互斥性：} 以“天”为最小不可分割的时间单位 [cite: 6]，玩家每天只能在“停留”、“行走”和“挖矿”中选择唯一一种作为当天的动作，不可将一天的时间拆分进行多种动作叠加。
\end{enumerate}

\section{模型的建立与求解}
\subsection{问题一模型建立}


\subsubsection{模型思路}

将沙漠地图抽象为无向图
\[
G=(V,E),
\]
其中，$V=\{1,2,\dots,n\}$ 表示地图中的区域节点集合，$E$ 表示节点之间的相邻关系。若节点 $i$ 与节点 $j$ 相邻，则玩家可以在一天内从 $i$ 移动到 $j$；若不相邻，则不能直接到达。

在此基础上，以“天”为决策阶段，对玩家每天的位置、行动、资源与资金变化进行描述。由于天气序列已知，问题一可建模为一个混合整数线性规划模型，以到达终点时的总资金最大为目标，对路径选择、行动安排和资源配置进行联合优化。



\subsubsection{变量定义}

\noindent\textbf{位置变量：}

定义二元变量
\[
x_{t,i}=
\begin{cases}
1, & \text{若第 } t \text{ 天结束时玩家位于节点 } i,\\
0, & \text{否则},
\end{cases}
\qquad t=0,1,\dots,T,\ i\in V.
\]

\noindent\textbf{行动变量：}

定义
\[
y_t^{\mathrm{stay}},\quad y_t^{\mathrm{move}},\quad y_t^{\mathrm{mine}}\in\{0,1\},
\qquad t=1,2,\dots,T,
\]
分别表示第 $t$ 天是否选择停留、移动或挖矿。由于每天只能执行一种主要行动，因此三者满足互斥关系。

\noindent\textbf{资源状态变量：}

定义
\[
W_t \ge 0,\qquad F_t \ge 0,\qquad M_t \ge 0,
\]
分别表示第 $t$ 天结束后玩家剩余的水量、食物量和资金。

\noindent\textbf{购买变量：}

定义
\[
u_t^w\ge 0,\qquad u_t^f\ge 0,
\]
分别表示第 $t$ 天购买的水量和食物量。其中，第 $0$ 天的购买为起点初始采购，其余天数若发生购买，则表示在村庄处的补给行为。

\subsubsection{目标函数}

题目规定，玩家到达终点后可将剩余的水和食物按基准价格的一半退回。因此，终点时的总资金应由剩余现金与剩余物资折现两部分组成。目标函数为
\[
\max Z=M_T+\frac{c_w}{2}W_T+\frac{c_f}{2}F_T.
\]

\subsubsection{约束条件}

\noindent\textbf{初始状态约束：}

第 $0$ 天玩家位于起点，且只能在起点进行一次初始购买，有
\[
x_{0,S}=1,
\]
\[
x_{0,i}=0,\qquad i\in V,\ i\neq S,
\]
\[
W_0=u_0^w,\qquad F_0=u_0^f,
\]
\[
M_0^{\prime}=M_0-c_wu_0^w-c_fu_0^f.
\]
同时，初始购买后需满足负重约束：
\[
g_wW_0+g_fF_0\le B.
\]

\noindent\textbf{位置唯一性约束：}

任意一天结束时，玩家只能位于一个节点：
\[
\sum_{i\in V}x_{t,i}=1,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{行动唯一性约束：}

每天只能执行一种主要行动：
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{move}}+y_t^{\mathrm{mine}}=1,\qquad t=1,2,\dots,T.
\]

\noindent\textbf{位置移动约束：}

若第 $t$ 天选择移动，则玩家只能从前一天所在节点移动到相邻节点；若选择停留或挖矿，则当天结束时位置不变。故有
\[
x_{t,j}\le x_{t-1,j}+\sum_{i:(i,j)\in E}x_{t-1,i},\qquad \forall j\in V,\ t=1,2,\dots,T.
\]

进一步地，可通过动作变量约束位置变化方式：
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{mine}}=1 \Longrightarrow x_{t,i}=x_{t-1,i},\qquad \forall i\in V,
\]
\[
y_t^{\mathrm{move}}=1 \Longrightarrow x_{t,i}\neq x_{t-1,i}\ \text{且仅能移动至相邻节点}.
\]

\noindent\textbf{天气约束：}

设第 $t$ 天天气记为 $\omega_t$。根据题意，若当天为沙暴，则玩家必须原地停留，不能移动。因此有
\[
\omega_t=\text{沙暴} \Longrightarrow y_t^{\mathrm{move}}=0.
\]

\noindent\textbf{挖矿约束：}

根据题目规则，只有在矿山停留时才可挖矿，且到达矿山当天不能挖矿。因此：
\[
y_t^{\mathrm{mine}}\le \sum_{i\in V_m}x_{t-1,i},\qquad t=1,2,\dots,T,
\]
即第 $t-1$ 天已经位于矿山时，第 $t$ 天才允许挖矿。此外，挖矿意味着当天位置不变。

\noindent\textbf{购买约束：}

起点只能在第 $0$ 天购买一次；对于 $t\ge 1$，只有当玩家位于村庄时才允许购买资源，因此可写为
\[
u_t^w+u_t^f \le L\sum_{i\in V_c}x_{t,i},\qquad t=1,2,\dots,T,
\]
其中 $L$ 为足够大的常数。该约束保证只有在村庄节点时购买量才可能为正。

\noindent\textbf{资源动态平衡约束：}

设第 $t$ 天在当前天气下停留时水和食物的基础消耗量分别为 $d_{w,t}$ 和 $d_{f,t}$。根据题意：

\begin{itemize}
    \item 停留消耗基础消耗量的 $1$ 倍；
    \item 移动消耗基础消耗量的 $2$ 倍；
    \item 挖矿消耗基础消耗量的 $3$ 倍。
\end{itemize}

因此，第 $t$ 天的实际消耗量分别为
\[
C_t^w=d_{w,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr),
\]
\[
C_t^f=d_{f,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr).
\]

相应的资源动态平衡方程为
\[
W_t=W_{t-1}-C_t^w+u_t^w,
\]
\[
F_t=F_{t-1}-C_t^f+u_t^f,
\qquad t=1,2,\dots,T.
\]

同时要求任意时刻资源非负：
\[
W_t\ge 0,\qquad F_t\ge 0,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{资金平衡约束：}

第 $0$ 天在起点购买物资时按基准价格扣除资金；第 $t\ge 1$ 天若在村庄购买物资，则按基准价格的 $2$ 倍支付；若第 $t$ 天挖矿，则获得基础收益 $R$。因此资金动态方程为
\[
M_t=M_{t-1}-2c_wu_t^w-2c_fu_t^f+Ry_t^{\mathrm{mine}},
\qquad t=1,2,\dots,T.
\]

同时要求任意时刻剩余资金非负：
\[
M_t\ge 0,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{负重约束：}

任意一天结束后，玩家携带的水和食物总质量不得超过负重上限：
\[
g_wW_t+g_fF_t\le B,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{终点约束：}

玩家必须在截止日期内到达终点，因此要求在最终时刻满足
\[
x_{T,D}=1.
\]

若进一步考虑“到达即结束”的情形，则可在程序求解中加入终点吸收状态，使玩家一旦到达终点后不再继续消耗资源。

\subsubsection{问题一的优化模型}

综上，问题一可归结为如下混合整数线性规划模型：
\[
\max Z=M_T+\frac{c_w}{2}W_T+\frac{c_f}{2}F_T
\]
\[
\text{s.t.}
\]
\[
x_{0,S}=1,\qquad x_{0,i}=0\ (i\neq S),
\]
\[
\sum_{i\in V}x_{t,i}=1,\qquad t=0,1,\dots,T,
\]
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{move}}+y_t^{\mathrm{mine}}=1,\qquad t=1,2,\dots,T,
\]
\[
x_{t,j}\le x_{t-1,j}+\sum_{i:(i,j)\in E}x_{t-1,i},\qquad \forall j\in V,
\]
\[
\omega_t=\text{沙暴} \Longrightarrow y_t^{\mathrm{move}}=0,
\]
\[
y_t^{\mathrm{mine}}\le \sum_{i\in V_m}x_{t-1,i},
\]
\[
u_t^w+u_t^f \le L\sum_{i\in V_c}x_{t,i},
\]
\[
W_t=W_{t-1}-d_{w,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr)+u_t^w,
\]
\[
F_t=F_{t-1}-d_{f,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr)+u_t^f,
\]
\[
M_t=M_{t-1}-2c_wu_t^w-2c_fu_t^f+Ry_t^{\mathrm{mine}},
\]
\[
g_wW_t+g_fF_t\le B,
\]
\[
W_t\ge 0,\qquad F_t\ge 0,\qquad M_t\ge 0,
\]
\[
x_{T,D}=1,
\]
\[
x_{t,i}\in\{0,1\},\quad y_t^{\mathrm{stay}},y_t^{\mathrm{move}},y_t^{\mathrm{mine}}\in\{0,1\},
\]
\[
u_t^w,u_t^f\in\mathbb{Z}_{\ge 0}.
\]

\subsubsection{求解思路}

由于问题一中的天气序列、地图结构及资源参数均已知，故上述模型是一个确定性优化模型。
求解时，先根据附件地图建立邻接关系，再构造位置变量、行动变量、购买变量以及资源和资金状态变量，
随后将目标函数与各类约束输入整数规划求解器进行求解。最终即可得到最优路径、每天的行动方案以及对应的物资购买策略。
从模型结构上看，该模型同时考虑了路径选择、资源配置与收益计算，能够统一描述玩家在已知天气条件下的最优穿越过程，因此适合作为问题一的一般模型。

\subsection{问题一模型求解结果}

利用Python中的PuLP库调用底层运筹学求解器，对上述混合整数线性规划（MILP）模型进行求解。根据模型输出结果，可得出两关在已知全局天气条件下的理论最优策略及对应的最大剩余资金。

\textbf{第一关求解结果：}  
在第一关的设定下，求解器寻找到的理论最优策略可保留最高资金为\textbf{10470元}。
最优路径策略为：玩家在第0天于起点购买\textbf{172箱水}和\textbf{333箱食物}，随后经

到达12号矿山，并在矿山进行\textbf{6日挖矿}以积累资金；

\section{问题二模型建立：仅知当天天气下的资金最优策略}

在问题一中，我们已经通过已知全局天气信息建立了最优路径模型。对于问题二，情况发生了变化：玩家仅能知晓当天的天气状况，对未来的天气变化无法预知。在这种条件下，我们需要使用**马尔可夫决策过程（MDP）**来建模。

\subsection{状态空间定义}

在每一天，玩家的状态可以用四元组 $(i, W, F, \omega)$ 来表示，其中：
\begin{itemize}
    \item $i$ 表示玩家所在的位置；
    \item $W$ 表示当前拥有的水量；
    \item $F$ 表示当前拥有的食物量；
    \item $\omega$ 表示当天的天气状态，$\omega \in \{\text{晴朗}, \text{高温}, \text{沙暴}\}$。
\end{itemize}

\subsection{决策变量与动作}

在每一天，玩家可以选择三种可能的动作选择：
\begin{itemize}
    \item 停留：在当前节点停留，消耗一定的资源（根据天气的不同消耗不同的水和食物）；
    \item 移动：玩家选择移动到相邻的节点，消耗水和食物的数量是停留的两倍；
    \item 挖矿：如果玩家在矿山节点，选择挖矿则消耗资源且获得挖矿收益。
\end{itemize}

\subsection{转移概率与奖励函数}

每个状态转移的奖励函数 $R(i, W, F, \omega, a)$ 包含了玩家根据当前状态和选择的动作所获得的即时收益。

- **停留**（Stay）：水和食物的消耗，若天气为沙暴，则无法行走，停留时只消耗水和食物。
- **行走**（Move）：水和食物的消耗，且消耗量为停留的两倍。
- **挖矿**（Mine）：挖矿可以获得资源，且消耗的是三倍水和食物。

\subsection{贝尔曼最优方程}

最优价值函数 $V_t(i, W, F, \omega)$ 满足贝尔曼最优方程：
\[
V_t(i, W, F, \omega) = \max_{a \in \{\text{Stay}, \text{Move}, \text{Mine}\}} \left[ R(i, W, F, \omega, a) + \mathbb{E}[V_{t+1}(i', W', F', \omega')]\right]
\]
其中，$\mathbb{E}[V_{t+1}]$ 表示下一天的期望价值，考虑到天气的不确定性，我们根据天气的先验概率计算该期望。

\subsection{动态规划与求解}

由于天气是随机的，因此每个状态的最优决策不仅依赖于当前状态，还需要考虑未来可能的天气变化。为了求解最优策略，我们使用动态规划中的逆向归纳法，从最后一天开始，逐步推导每一天的最优价值函数。

具体地，我们通过从终点的已知价值开始，向前推导每个状态的最优价值，直到第 0 天。最终，我们得到在每一天的每个状态下，最优的行动策略。

\subsection{模型总结}

该模型的核心是使用马尔可夫决策过程来处理不确定的天气状态，动态调整玩家的行为策略，以最大化最终的资金保留。在实际求解过程中，我们将通过逆向动态规划计算每个状态下的最优行动，并在游戏进行过程中根据实际天气调整策略。

\subsubsection{模型求解结果与策略解释}

基于上述模型，我们采用逆向动态规划方法逐日计算价值函数张量。最终可得到以下两类结果：

\textbf{第三关求解结果分析：} 在第三关的设定下，由于天气状态不再是全局确定已知，而是依据先验概率随机出现，
因此我们采用有限时域马尔可夫决策过程与逆向动态规划方法进行求解。
计算结果表明，在该随机环境下，玩家的最大期望最终保留资金为 \textbf{12730 元}。
最优策略表现出明显的“前期推进—中途补给—矿山集中获利—末期冲刺终点”的阶段性特征：
玩家在第 0 天于起点购买 130 箱水和 405 箱食物，并充分利用初始负重上限，为后续长距离推进奠定资源基础；
随后沿 2→10→19→20→28→37→38→39 的路径前进，在前期通过适当停留平衡天气变化带来的资源消耗压力，
并于第 10 天在 39 号村庄补充 211 箱水，以保障后续进入矿山区域的续航能力。
此后玩家经 47→55 到达 55 号矿山，在第 14 天至第 18 天连续挖矿 5 天获取第一阶段收益；
随后于第 19 天前往 62 号村庄补充 174 箱水和 86 箱食物，再于第 20 天返回 55 号矿山，于第 21 天至第 28 天继续挖矿 8 天，
从而在满足终点时限约束的前提下尽可能延长高收益作业时间。
最终玩家于第 29 天经 63 号节点前进，并于第 30 天安全抵达 64 号终点，且水与食物恰好耗尽，最终资金达到 \textbf{12730 元}。
这一结果说明，在随机天气背景下，最优策略并非简单追求最短路径，而是在资源消耗、补给时机、挖矿收益与终点可达性之间实现了全局最优平衡。


与问题一相比，问题二不再输出一条唯一的静态最优路径，而是输出一个\textbf{面向随机天气变化的动态反馈策略}。
该策略并不依赖对未来天气的准确预知，而是根据当天已知天气信息实时调整行动，因此更符合不确定环境下的实际决策逻辑。
模型结果表明，该方法能够在天气随机扰动下兼顾生存性、可达性与盈利性，为玩家提供具有期望最优意义的全过程决策依据。

\section{模型评价与改进}

\subsection{模型的优点}

\begin{itemize}
    \item \textbf{全局最优性与极强的数学严密性：} 针对确定性环境下的问题一，本文摒弃了容易陷入局部最优的启发式搜索（如贪心算法、回溯法），利用混合整数线性规划（MILP）进行全局精确求解[cite: 47, 74]。针对问题三的拥挤博弈，摒弃了主观的“碰撞计分法”，严格基于 $2k$ 倍资源消耗法则计算真实资金支付矩阵，确保了模型输出的策略为严密的全局最优解与混合策略纳什均衡。
    \item \textbf{巧妙的降维技巧与极高的计算效率：} 在求解问题二的马尔可夫决策过程（MDP）时，面对第四关庞大的物资组合与村庄随时购买机制，若采用常规前向搜索，复杂度将呈指数级爆炸[cite: 53]。本文利用状态价值的连续性，创新性地提出了“村庄物资价值的逆向传播”降维优化算法，将 $O(W^2 \times F^2)$ 的复杂度降维至 $O(W+F)$，利用 NumPy 张量运算在数秒内即可完成三维状态空间的值迭代。
    \item \textbf{强大的策略泛化能力：} 在问题二与问题四的求解中，模型输出的并非一条死板的静态路径，而是一个完整的最优策略函数 $\pi^*(S_t)$[cite: 50, 225]。无论玩家在游戏进行中遭遇何种极端的随机天气序列，只需依据当前的时空与物资状态查询价值张量，即可动态获取当前最优动作，具有极强的鲁棒性与实战指导意义。
    \item \textbf{深刻的经济学与博弈论洞察：} 在第五关的多人博弈分析中，模型精准捕捉到了“前往矿山的边际收益远高于绕路避险的边际成本”这一经济学本质。通过单纯形法求解出 100\% 直接冲刺的纯策略纳什均衡，深刻揭示了零和竞争下“囚徒困境”与“内卷”的必然性，展现了极高的模型洞察力。
\end{itemize}

\subsection{模型的缺点}

\begin{itemize}
    \item \textbf{对先验概率的强依赖性：} 在求解马尔可夫决策过程与动态期望支付矩阵时，未来天气的期望价值高度依赖于输入的先验概率分布（如设定晴天概率为 0.44，高温为 0.56）[cite: 138, 139]。若游戏底层的真实天气生成机制发生偏移（例如出现连续极端天气的长尾效应），依据固定期望制定的策略可能会带来一定的资金折损风险。
    \item \textbf{多智能体动态博弈的滚动时域局限：} 在第六关（随机天气下的多玩家博弈）中，由于联合状态空间过于庞大，我们采用了“每日滚动构建期望支付矩阵并求解纳什均衡”的贪心降维策略[cite: 57]。这种基于单步或有限步前瞻的动态规划近似，虽然计算高效，但在理论上属于次优解（Sub-optimal），难以完全等价于全局最优的马尔可夫完美均衡（Markov Perfect Equilibrium）。
    \item \textbf{绝对理性人假设的局限性：} 模型假设所有竞争对手均是算力无限且绝对理性的[cite: 63]，始终以资金最大化为目标执行纳什均衡策略。然而在真实的交互博弈中，对手可能存在“有限理性（Bounded Rationality）”或强烈的风险厌恶倾向，这种行为偏差可能会使实际游戏结果偏离模型的理论预测。
\end{itemize}

\subsection{模型的改进}

\begin{itemize}
    \item \textbf{引入多智能体深度强化学习（MARL）：} 针对第六关中状态空间爆炸与不完全信息（局部视野）的痛点，后续研究可引入多智能体近端策略优化算法（MAPPO）或多智能体深度确定性策略梯度（MADDPG）。采用“集中式训练、去中心化执行（CTDE）”架构，让智能体在百万次的对抗性自我博弈（Self-Play）中利用神经网络拟合高维策略空间，从而摆脱对人工提取支付矩阵的依赖。
    \item \textbf{引入鲁棒优化与风险度量（CVaR）：} 针对概率依赖问题，可将当前的“风险中性（期望值最大化）”模型升级为“风险厌恶”模型。引入条件风险价值（Conditional Value at Risk, CVaR）或分布鲁棒优化（Distributionally Robust Optimization），使得玩家在制定策略时，不仅追求平均收益最高，还能最大程度保证在最恶劣天气序列（如连续 10 天高温/沙暴）下的生存下限。
    \item \textbf{基于贝叶斯推断的动态概率更新：} 摒弃全局固定的天气先验概率，引入隐马尔可夫模型（HMM）或贝叶斯更新机制。玩家在每天前行时，可根据过去几天的天气历史（观测序列），动态修正对未来天气的后验概率估计，从而做出更加敏锐和贴合实际环境的自适应决策。
\end{itemize}
