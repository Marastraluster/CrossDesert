\section*{\zihao{4}\mbox{题目：基于状态空间搜索与MILP的单/多玩家沙漠生存博弈寻优}}
\vspace{1.5em}

\section*{摘要}
本文以“穿越沙漠”问题为对象，研究在资源约束、天气扰动与多玩家竞争条件下的路径规划与策略优化问题。围绕统一的“状态—决策—收益”框架，本文将位置、资源、天气与交互关系表示为系统状态，以停留、行走、挖矿和补给作为可行动作，并以终点清算资金最大化作为总体目标。针对不同信息结构，分别建立三类模型：在全局天气已知情形下，构建混合整数线性规划模型求解确定性最优策略；在当天天气已知、未来天气随机情形下，构建有限时域马尔可夫决策过程并采用逆向动态规划求解；在多玩家耦合情形下，构建滚动反馈的动态博弈评估模型，并通过蒙特卡洛模拟检验策略稳定性。

结果表明：确定性场景下模型能够给出可复现的最优路径与补给方案；随机场景下动态策略相较静态路径具有更好的环境适应性；多玩家场景下所构造策略在存活率、到达率与团队收益之间取得了较好的综合平衡。本文的主要贡献在于给出一套可扩展的统一建模范式，使单人确定性优化、单人随机决策与多人竞争协同能够在同一符号体系下进行描述、求解与比较。

\vspace{1.2em}
\noindent\textbf{关键词：} 混合整数线性规划；马尔可夫决策过程；动态博弈；滚动优化；蒙特卡洛模拟

\vspace{1.5em}


\section{问题重述}

\subsection{问题背景与优化目标}
本项目考察的是一个带有资源调度与路径规划性质的沙漠生存游戏。玩家需要凭借一张地图，从起点出发，利用初始资金购买特定数量的水和食物，途经不同的地形区域（如普通节点、矿山、村庄），最终在规定的截止日期或之前抵达终点。本题的优化目标是：在确保玩家成功存活并按时到达终点的前提下，通过合理规划行进路线与物资调度，使得游戏结束时\textbf{保留尽可能多的资金}。

\subsection{统一建模主线}
为保证全文叙述具有统一性，本文不将各问拆分为彼此孤立的算法题，而是将其统一抽象为同一套“时空状态—动作决策—资源演化—终端清算”的闭环模型。具体而言，系统状态由位置、资源、天气及多人交互状态共同刻画，动作决策由停留、行走、挖矿与补给构成，状态转移由天气、地形与规则约束共同决定，终端价值则统一取为剩余资金及物资折现值。问题一、问题二与问题三的区别，只在于天气是否全局已知、未来是否存在随机性，以及是否引入多玩家耦合效应；因此后文的建模、推导与求解都尽量围绕同一套符号体系展开。

\subsection{核心规则与约束条件}
玩家在游戏中的决策受到以下多重物理与规则约束：
\begin{itemize}
    \item \textbf{负重与生存约束：} 每天玩家拥有的水和食物质量之和不能超过负重上限。若未到达终点而水或食物已耗尽，视为游戏失败。
    \item \textbf{天气与环境影响：} 每天的天气为“晴朗”、“高温”、“沙暴”三种状况之一。沙暴日玩家必须在原地停留。
    \item \textbf{节点交互系统：} 第0天可在起点处用初始资金以基准价格购买水和食物，但不能多次购买；在村庄停留时可用剩余资金随时购买水和食物，每箱价格为基准价格的2倍；在矿山停留可通过挖矿获得资金，但到达矿山当天不能挖矿；到达终点后可退回剩余的水和食物，每箱退回价格为基准价格的一半。
    \item \textbf{消耗倍率：} 玩家在原地停留一天消耗的资源数量为基础消耗量，行走一天消耗的资源数量为基础消耗量的2倍，挖矿消耗的资源数量为基础消耗量的3倍。
\end{itemize}

\subsection{问题一的具体任务}
问题一设定为\textbf{假设只有一名玩家}，且属于全局确定性环境（即\textbf{在整个游戏时段内每天天气状况事先全部已知}）。任务要求建立数学模型，给出一般情况下玩家的最优策略，并利用该模型求解附件中的“第一关”和“第二关”。
\section{问题分析}

\subsection{问题一的分析}
问题一要求在全局天气信息完全已知的情况下，为单名玩家制定使得最终保留资金最大化的行动策略。针对该问题，我们从以下四个维度展开分析：

\subsubsection{问题特征分析}
本问题属于典型的\textbf{确定性多阶段离散状态转移问题}。在“第一关”与“第二关”中，玩家在整个游戏周期内面对的天气序列、地图拓扑与基础价格均已知且固定，因此可将问题表述为时间扩展网络上的最优路径规划问题。

\subsubsection{关键变量界定}
为了精确描述玩家的状态转移，需要追踪的核心变量包括：
\begin{itemize}
    \item \textbf{时空变量：} 当前所处天数 $t$ 以及玩家当天所在的地图节点 $i$。
    \item \textbf{决策变量：} 玩家每日采取的具体动作（停留、行走或挖矿），以及在特定节点（起点或村庄）的物资购买量。
    \item \textbf{状态变量：} 每日结算后的剩余资金量 $M_t$、水量 $W_t$ 与食物量 $F_t$。
\end{itemize}

\subsubsection{模型难点剖析}
本题的主要难点在于\textbf{多重约束的耦合}以及\textbf{逻辑条件的线性化}：
\begin{itemize}
    \item \textbf{负重与消耗约束：} 负重上限（1200 kg）限制了玩家可携带的资源总量，而天气与动作类型共同决定每日消耗水平，因此需要在续航能力与补给成本之间进行权衡。
    \item \textbf{规则约束的表达：} 沙暴日禁止移动、村庄购买价格翻倍、到达矿山当天不能挖矿等规则均带有条件性，需要通过变量和约束式加以刻画。
\end{itemize}

\subsubsection{可行求解思路}
鉴于时间跨度和节点规模均处于可控范围内，且问题一具备全局信息确定性，本文采用\textbf{混合整数线性规划（MILP）}求解该确定性特例。具体做法是以0-1变量刻画每日位置与动作选择，并利用\textbf{大 M 法（Big-M Method）}将购买、挖矿等条件判断转化为线性约束，进而在终点资金最大化目标下求得全局最优解。
\subsection{问题二的分析}

问题二是在前述母模型上放宽天气信息假设：玩家仅能获知当天天气，未来天气则服从给定先验分布。状态变量、动作变量、资源演化和终端清算规则仍沿用统一框架，但决策者的可见信息由完整天气序列缩减为当前时点信息，因此问题转化为随机动态决策问题。

\begin{itemize}
    \item \textbf{决策范式的变化：} 由于未来天气具有随机性，玩家不能在第0天预先确定一条固定路径，而需要根据已知状态动态调整当日动作。因此，问题的目标由“求一条最优路径”转化为“求一个最优策略”，即在任意日期、位置、资源和天气状态下选择期望收益最大的动作。
    \item \textbf{模型映射与求解：} 这一结构符合\textbf{马尔可夫决策过程（Markov Decision Process, MDP）}的建模方式。为求解最大期望终值，本文采用逆向动态规划，并由贝尔曼最优方程递推价值函数。对于村庄补给这一连续决策，进一步结合状态价值的单调性进行简化处理，以降低计算复杂度。
\end{itemize}

\subsection{问题三的分析}

问题三在统一状态转移机制上进一步引入多玩家耦合。与前两问相比，变化主要体现在同一时刻存在多个决策主体，各主体之间通过拥挤消耗、收益分摊和价格调整产生相互影响，因此问题可表述为多主体动态博弈。

\begin{itemize}
    \item \textbf{第五关分析（完全信息静态博弈）：} 第五关天气事先已知，且行动方案不可更改。由于多人在相同路径和节点上存在拥挤效应，该问题可视为一个\textbf{拥挤博弈（Congestion Game）}。因此，需要构造覆盖不同时间安排与路径选择的策略集合，并通过逐日碰撞模拟计算各策略对应的收益矩阵，再求解相应的均衡策略。
    
    \item \textbf{第六关分析（不完全信息动态博弈）：} 第六关结合随机天气与多玩家竞争因素，玩家仅能获得局部状态信息，因此更适合建模为\textbf{随机马尔可夫博弈（Stochastic Markov Games）}。在这一框架下，需要基于当日可观测信息、天气概率分布及其他玩家状态，逐日构造期望收益并进行滚动决策。
\end{itemize}
\section{模型假设}
鉴于题目中已经给出了较为详尽的物理约束与游戏规则，为了确保数学模型的严谨性与求解的可行性，我们对实际游戏过程中的部分隐性条件进行如下合理假设：

\begin{enumerate}
    \item \textbf{假设玩家具有完全理性：} 玩家在整个游戏过程中具有全局最优视野，始终以最终保留资金最大化为唯一目标进行决策，不存在操作失误或非理性的随意探索行为。
    \item \textbf{假设物资交易行为瞬间完成：} 玩家在起点或村庄购买水和食物时，交易过程不额外消耗当天的行动时间。同时假设村庄的物资储备无限大，不受其他因素影响。
    \item \textbf{假设资源在储存与携带过程中无额外非线性损耗：} 水和食物除了按照每日天气和行动状态的设定被消耗外，不会因天气炎热、沙暴环境或存储时间过长而发生自然蒸发、变质或意外丢失。
    \item \textbf{假设行动状态具有时间离散性与动作互斥性：} 以“天”为最小不可分割的时间单位，玩家每天只能在“停留”、“行走”和“挖矿”中选择唯一一种作为当天的动作，不可将一天的时间拆分进行多种动作叠加。
\end{enumerate}

\section{模型的建立与求解}
本节不将三问视为彼此割裂的独立题目，而是统一为同一套“时空状态—动作决策—资源演化—终端清算”的母模型：问题一对应全局信息已知下的确定性最优控制，问题二对应信息受限下的随机动态决策，问题三对应在同一状态转移机制上叠加多人耦合后的动态博弈。后文所有符号、约束与目标函数都尽量沿用同一套表达，只在信息可见性和交互规则上做最小必要扩展。

\subsection{问题一模型建立}


\subsubsection{模型思路}

将沙漠地图抽象为无向图
\[
G=(V,E),
\]
其中，$V=\{1,2,\dots,n\}$ 表示地图中的区域节点集合，$E$ 表示节点之间的相邻关系。若节点 $i$ 与节点 $j$ 相邻，则玩家可以在一天内从 $i$ 移动到 $j$；若不相邻，则不能直接到达。

在此基础上，以“天”为决策阶段，对玩家每天的位置、行动、资源与资金变化进行描述。由于天气序列已知，问题一可建模为一个混合整数线性规划模型，以到达终点时的总资金最大为目标，对路径选择、行动安排和资源配置进行联合优化。



\subsubsection{变量定义}

\noindent\textbf{位置变量：}

定义二元变量
\[
x_{t,i}=
\begin{cases}
1, & \text{若第 } t \text{ 天结束时玩家位于节点 } i,\\
0, & \text{否则},
\end{cases}
\qquad t=0,1,\dots,T,\ i\in V.
\]

\noindent\textbf{行动变量：}

定义
\[
y_t^{\mathrm{stay}},\quad y_t^{\mathrm{move}},\quad y_t^{\mathrm{mine}}\in\{0,1\},
\qquad t=1,2,\dots,T,
\]
分别表示第 $t$ 天是否选择停留、移动或挖矿。由于每天只能执行一种主要行动，因此三者满足互斥关系。

\noindent\textbf{资源状态变量：}

定义
\[
W_t \ge 0,\qquad F_t \ge 0,\qquad M_t \ge 0,
\]
分别表示第 $t$ 天结束后玩家剩余的水量、食物量和资金。

\noindent\textbf{购买变量：}

定义
\[
u_t^w\ge 0,\qquad u_t^f\ge 0,
\]
分别表示第 $t$ 天购买的水量和食物量。其中，第 $0$ 天的购买为起点初始采购，其余天数若发生购买，则表示在村庄处的补给行为。

\subsubsection{目标函数}

题目规定，玩家到达终点后可将剩余的水和食物按基准价格的一半退回。因此，终点时的总资金应由剩余现金与剩余物资折现两部分组成。目标函数为
\[
\max Z=M_T+\frac{c_w}{2}W_T+\frac{c_f}{2}F_T.
\]

\subsubsection{约束条件}

\noindent\textbf{初始状态约束：}

第 $0$ 天玩家位于起点，且只能在起点进行一次初始购买，有
\[
x_{0,S}=1,
\]
\[
x_{0,i}=0,\qquad i\in V,\ i\neq S,
\]
\[
W_0=u_0^w,\qquad F_0=u_0^f,
\]
\[
M_0^{\prime}=M_0-c_wu_0^w-c_fu_0^f.
\]
同时，初始购买后需满足负重约束：
\[
g_wW_0+g_fF_0\le B.
\]

\noindent\textbf{位置唯一性约束：}

任意一天结束时，玩家只能位于一个节点：
\[
\sum_{i\in V}x_{t,i}=1,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{行动唯一性约束：}

每天只能执行一种主要行动：
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{move}}+y_t^{\mathrm{mine}}=1,\qquad t=1,2,\dots,T.
\]

\noindent\textbf{位置移动约束：}

若第 $t$ 天选择移动，则玩家只能从前一天所在节点移动到相邻节点；若选择停留或挖矿，则当天结束时位置不变。故有
\[
x_{t,j}\le x_{t-1,j}+\sum_{i:(i,j)\in E}x_{t-1,i},\qquad \forall j\in V,\ t=1,2,\dots,T.
\]

进一步地，可通过动作变量约束位置变化方式：
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{mine}}=1 \Longrightarrow x_{t,i}=x_{t-1,i},\qquad \forall i\in V,
\]
\[
y_t^{\mathrm{move}}=1 \Longrightarrow x_{t,i}\neq x_{t-1,i}\ \text{且仅能移动至相邻节点}.
\]

\noindent\textbf{天气约束：}

设第 $t$ 天天气记为 $\omega_t$。根据题意，若当天为沙暴，则玩家必须原地停留，不能移动。因此有
\[
\omega_t=\text{沙暴} \Longrightarrow y_t^{\mathrm{move}}=0.
\]

\noindent\textbf{挖矿约束：}

根据题目规则，只有在矿山停留时才可挖矿，且到达矿山当天不能挖矿。因此：
\[
y_t^{\mathrm{mine}}\le \sum_{i\in V_m}x_{t-1,i},\qquad t=1,2,\dots,T,
\]
即第 $t-1$ 天已经位于矿山时，第 $t$ 天才允许挖矿。此外，挖矿意味着当天位置不变。

\noindent\textbf{购买约束：}

起点只能在第 $0$ 天购买一次；对于 $t\ge 1$，只有当玩家位于村庄时才允许购买资源，因此可写为
\[
u_t^w+u_t^f \le L\sum_{i\in V_c}x_{t,i},\qquad t=1,2,\dots,T,
\]
其中 $L$ 为足够大的常数。该约束保证只有在村庄节点时购买量才可能为正。

\noindent\textbf{资源动态平衡约束：}

设第 $t$ 天在当前天气下停留时水和食物的基础消耗量分别为 $d_{w,t}$ 和 $d_{f,t}$。根据题意：

\begin{itemize}
    \item 停留消耗基础消耗量的 $1$ 倍；
    \item 移动消耗基础消耗量的 $2$ 倍；
    \item 挖矿消耗基础消耗量的 $3$ 倍。
\end{itemize}

因此，第 $t$ 天的实际消耗量分别为
\[
C_t^w=d_{w,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr),
\]
\[
C_t^f=d_{f,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr).
\]

相应的资源动态平衡方程为
\[
W_t=W_{t-1}-C_t^w+u_t^w,
\]
\[
F_t=F_{t-1}-C_t^f+u_t^f,
\qquad t=1,2,\dots,T.
\]

同时要求任意时刻资源非负：
\[
W_t\ge 0,\qquad F_t\ge 0,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{资金平衡约束：}

第 $0$ 天在起点购买物资时按基准价格扣除资金；第 $t\ge 1$ 天若在村庄购买物资，则按基准价格的 $2$ 倍支付；若第 $t$ 天挖矿，则获得基础收益 $R$。因此资金动态方程为
\[
M_t=M_{t-1}-2c_wu_t^w-2c_fu_t^f+Ry_t^{\mathrm{mine}},
\qquad t=1,2,\dots,T.
\]

同时要求任意时刻剩余资金非负：
\[
M_t\ge 0,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{负重约束：}

任意一天结束后，玩家携带的水和食物总质量不得超过负重上限：
\[
g_wW_t+g_fF_t\le B,\qquad t=0,1,\dots,T.
\]

\noindent\textbf{终点约束：}

玩家必须在截止日期内到达终点，因此要求在最终时刻满足
\[
x_{T,D}=1.
\]

若进一步考虑“到达即结束”的情形，则可在程序求解中加入终点吸收状态，使玩家一旦到达终点后不再继续消耗资源。

\subsubsection{问题一的优化模型}

综上，问题一可归结为如下混合整数线性规划模型：
\[
\max Z=M_T+\frac{c_w}{2}W_T+\frac{c_f}{2}F_T
\]
\[
\text{s.t.}
\]
\[
x_{0,S}=1,\qquad x_{0,i}=0\ (i\neq S),
\]
\[
\sum_{i\in V}x_{t,i}=1,\qquad t=0,1,\dots,T,
\]
\[
y_t^{\mathrm{stay}}+y_t^{\mathrm{move}}+y_t^{\mathrm{mine}}=1,\qquad t=1,2,\dots,T,
\]
\[
x_{t,j}\le x_{t-1,j}+\sum_{i:(i,j)\in E}x_{t-1,i},\qquad \forall j\in V,
\]
\[
\omega_t=\text{沙暴} \Longrightarrow y_t^{\mathrm{move}}=0,
\]
\[
y_t^{\mathrm{mine}}\le \sum_{i\in V_m}x_{t-1,i},
\]
\[
u_t^w+u_t^f \le L\sum_{i\in V_c}x_{t,i},
\]
\[
W_t=W_{t-1}-d_{w,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr)+u_t^w,
\]
\[
F_t=F_{t-1}-d_{f,t}\bigl(y_t^{\mathrm{stay}}+2y_t^{\mathrm{move}}+3y_t^{\mathrm{mine}}\bigr)+u_t^f,
\]
\[
M_t=M_{t-1}-2c_wu_t^w-2c_fu_t^f+Ry_t^{\mathrm{mine}},
\]
\[
g_wW_t+g_fF_t\le B,
\]
\[
W_t\ge 0,\qquad F_t\ge 0,\qquad M_t\ge 0,
\]
\[
x_{T,D}=1,
\]
\[
x_{t,i}\in\{0,1\},\quad y_t^{\mathrm{stay}},y_t^{\mathrm{move}},y_t^{\mathrm{mine}}\in\{0,1\},
\]
\[
u_t^w,u_t^f\in\mathbb{Z}_{\ge 0}.
\]

\subsubsection{求解思路}

由于问题一中的天气序列、地图结构及资源参数均已知，故上述模型是一个确定性优化模型。
求解时，先根据附件地图建立邻接关系，再构造位置变量、行动变量、购买变量以及资源和资金状态变量，
随后将目标函数与各类约束输入整数规划求解器进行求解。最终即可得到最优路径、每天的行动方案以及对应的物资购买策略。
从模型结构上看，该模型同时考虑了路径选择、资源配置与收益计算，能够统一描述玩家在已知天气条件下的最优穿越过程，因此适合作为问题一的一般模型。

\textbf{第一关求解结果：}  
在第一关的设定下，求解器寻找到的理论最优策略可保留最高资金为\textbf{10470元}。
最优路径策略为：玩家在第0天于起点购买\textbf{172箱水}和\textbf{333箱食物}，随后经

\subsection{问题二：仅知道当天天气下的最优策略模型}

对于问题二，玩家在第 $t$ 天开始时仅知道当天的天气状况，而无法预先获知未来完整天气序列，因此该问题不再属于全局确定的静态规划问题，而应视为带有随机扰动的多阶段动态决策问题。为此，本文将其建模为有限时域马尔可夫决策过程，并以终点处的期望剩余资金最大化为目标，建立如下动态优化模型。

\subsubsection*{1. 状态变量与决策变量}

设第 $t$ 天系统状态为
\[
S_t=(L_t,W_t,F_t,M_t,T_t),
\]
其中：
\begin{itemize}
    \item $L_t$ 表示第 $t$ 天结束时玩家所在区域；
    \item $W_t$ 表示第 $t$ 天结束时剩余水量（箱）；
    \item $F_t$ 表示第 $t$ 天结束时剩余食物量（箱）；
    \item $M_t$ 表示第 $t$ 天结束时剩余资金（元）；
    \item $T_t$ 表示第 $t$ 天观测到的天气状态。
\end{itemize}

设天气状态空间为
\[
\Theta=\{1,2,3\},
\]
其中
\[
T_t=
\begin{cases}
1, & \text{晴朗},\\
2, & \text{高温},\\
3, & \text{沙暴}.
\end{cases}
\]

设第 $t$ 天的决策变量为
\[
D_t\in \mathcal{A}(S_t),
\]
其中
\[
\mathcal{A}(S_t)=\{1,2,3\},
\]
并规定
\[
D_t=
\begin{cases}
1, & \text{停留},\\
2, & \text{行走},\\
3, & \text{挖矿}.
\end{cases}
\]

此外，若玩家在村庄购买资源，则记第 $t$ 天购买的水和食物数量分别为
\[
U_t,\quad V_t.
\]

\subsubsection*{2. 天气转移与消耗函数}

由于玩家只能观察当天天气，因此未来天气视为随机变量。设天气满足马尔可夫性，则有
\[
\Pr(T_{t+1}\mid T_t,T_{t-1},\dots,T_1)=\Pr(T_{t+1}\mid T_t).
\]

记天气转移概率为
\[
p_{ij}=\Pr(T_{t+1}=j\mid T_t=i),\qquad i,j\in\Theta.
\]

设晴朗、高温、沙暴三种天气下单位基础耗水量分别为 $X_W^{1},X_W^{2},X_W^{3}$，单位基础耗食量分别为 $X_F^{1},X_F^{2},X_F^{3}$，则有
\[
X_W(T_t)=X_W^{T_t},\qquad X_F(T_t)=X_F^{T_t}.
\]

由题意可知，不同行为对应不同消耗倍率，定义倍率函数
\[
\beta(D_t)=
\begin{cases}
1, & D_t=1,\\
2, & D_t=2,\\
3, & D_t=3.
\end{cases}
\]

因此，第 $t$ 天的实际耗水量与耗食量分别为
\[
C_t^W=\beta(D_t)\,X_W(T_t),
\qquad
C_t^F=\beta(D_t)\,X_F(T_t).
\]

\subsubsection*{3. 状态转移方程}

若第 $t$ 天在村庄进行了补给，则下一时刻资源状态满足
\[
W_{t+1}=W_t+U_t-C_t^W,
\]
\[
F_{t+1}=F_t+V_t-C_t^F.
\]

设水和食物的基准价格分别为 $G_W,G_F$，村庄购买价格为基准价格的 2 倍；若第 $t$ 天挖矿，则获得基础收益 $R$。定义挖矿指示函数
\[
I(D_t)=
\begin{cases}
1, & D_t=3,\\
0, & D_t\neq 3.
\end{cases}
\]

则资金状态转移满足
\[
M_{t+1}=M_t+I(D_t)R-2G_WU_t-2G_FV_t.
\]

位置状态转移满足
\[
L_{t+1}=
\begin{cases}
L_t, & D_t=1,\\
L_t, & D_t=3,\\
j,\quad j\in \mathcal{N}(L_t), & D_t=2,
\end{cases}
\]
其中 $\mathcal{N}(L_t)$ 表示节点 $L_t$ 的相邻区域集合。

因此，可将系统状态转移记为
\[
S_{t+1}=f(S_t,D_t,T_{t+1}).
\]

\subsubsection*{4. 单阶段收益与终端收益}

第 $t$ 天的单阶段收益主要由挖矿收益与村庄补给成本构成，因此定义单阶段收益函数为
\[
R_t(S_t,D_t)=I(D_t)R-2G_WU_t-2G_FV_t.
\]

当玩家到达终点时，剩余水和食物可以按基准价格的一半折价回收，因此终端收益定义为
\[
H_T=M_T+\frac{1}{2}G_WW_T+\frac{1}{2}G_FF_T.
\]

于是，问题二的优化目标为
\[
\max\ \mathbb{E}(H_T).
\]

\subsubsection*{5. 价值函数与贝尔曼方程}

记价值函数 $V_t(S_t)$ 为：从第 $t$ 天开始，当系统处于状态 $S_t$ 时，在最优策略下所能获得的最大期望终端收益，则终端条件为
\[
V_T(S_T)=M_T+\frac{1}{2}G_WW_T+\frac{1}{2}G_FF_T.
\]

对于任意 $t=1,2,\dots,T-1$，有如下贝尔曼递推方程：
\[
V_t(S_t)
=
\max_{D_t\in\mathcal{A}(S_t)}
\left\{
R_t(S_t,D_t)
+\sum_{j\in\Theta}\Pr(T_{t+1}=j\mid T_t)\,V_{t+1}(S_{t+1})
\right\}.
\]

其中，
\[
S_{t+1}=f(S_t,D_t,T_{t+1}).
\]

因此，初始状态下的最大期望收益可表示为
\[
\mathbb{E}(H_T)=\sum_{S_1\in\mathcal{S}}\Pr(S_1)V_1(S_1).
\]

\subsubsection*{6. 约束条件}

根据游戏规则，有如下约束条件：

\textbf{（1）负重约束}
\[
3W_t+2F_t\le H_{\max},\qquad \forall t.
\]

\textbf{（2）非负约束}
\[
W_t\ge 0,\qquad F_t\ge 0,\qquad M_t\ge 0,\qquad \forall t.
\]

\textbf{（3）沙暴停留约束}
\[
T_t=3\Rightarrow D_t\neq 2.
\]

\textbf{（4）挖矿位置约束}
\[
D_t=3\Rightarrow L_t\in \mathcal{M},
\]
其中 $\mathcal{M}$ 为矿山节点集合。

\textbf{（5）到达矿山当天不能挖矿}
\[
D_t=3\Rightarrow L_{t-1}=L_t.
\]

\textbf{（6）购买位置约束}
\[
U_t>0\ \text{or}\ V_t>0 \Rightarrow L_t\in \mathcal{V},
\]
其中 $\mathcal{V}$ 为村庄节点集合。

\textbf{（7）路径可达约束}
\[
L_{t+1}\in \mathcal{N}(L_t)\cup\{L_t\}.
\]

\textbf{（8）起终点约束}
\[
L_0=L_{\mathrm{start}},\qquad L_T=L_{\mathrm{end}}.
\]

\textbf{（9）时间约束}
\[
t\le T_{\max}.
\]

\subsubsection*{7. 问题二的统一符号模型表示}

综上，问题二可统一表示为如下有限时域随机动态规划模型：
\[
\max\ \mathbb{E}(H_T)
\]

\[
\mathrm{s.t.}\quad
\left\{
\begin{aligned}
V_t(S_t)
&=
\max_{D_t\in\mathcal{A}(S_t)}
\left\{
R_t(S_t,D_t)
+\sum_{j\in\Theta}\Pr(T_{t+1}=j\mid T_t)V_{t+1}(S_{t+1})
\right\},
\\[4pt]
V_T(S_T)
&=
M_T+\frac12 G_WW_T+\frac12 G_FF_T,
\\[4pt]
S_t&=(L_t,W_t,F_t,M_t,T_t),
\\[4pt]
S_{t+1}&=f(S_t,D_t,T_{t+1}),
\\[4pt]
C_t^W&=\beta(D_t)X_W(T_t),
\\[4pt]
C_t^F&=\beta(D_t)X_F(T_t),
\\[4pt]
W_{t+1}&=W_t+U_t-C_t^W,
\\[4pt]
F_{t+1}&=F_t+V_t-C_t^F,
\\[4pt]
M_{t+1}&=M_t+I(D_t)R-2G_WU_t-2G_FV_t,
\\[4pt]
3W_t+2F_t&\le H_{\max},
\\[4pt]
W_t,\;F_t,\;M_t&\ge 0,
\\[4pt]
L_{t+1}&\in\mathcal{N}(L_t)\cup\{L_t\},
\\[4pt]
T_t&\in\Theta=\{1,2,3\},
\\[4pt]
D_t&\in\{1,2,3\},
\\[4pt]
T_t=3&\Rightarrow D_t\neq 2,
\\[4pt]
D_t=3&\Rightarrow L_t\in\mathcal{M},\ L_{t-1}=L_t,
\\[4pt]
U_t>0\ \text{or}\ V_t>0&\Rightarrow L_t\in\mathcal{V},
\\[4pt]
L_0&=L_{\mathrm{start}},\quad L_T=L_{\mathrm{end}},
\\[4pt]
t&\le T_{\max}.
\end{aligned}
\right.
\]

由此，问题二可视为一个“状态—决策—随机天气—状态更新”的闭环动态优化问题。玩家需依据当天实际天气、当前位置及剩余资源作出决策，从而使终点处的期望剩余资金最大。

\subsubsection{模型求解结果与策略解释}

基于上述模型，我们采用逆向动态规划方法逐日计算价值函数张量。最终可得到以下两类结果：

\textbf{第三关求解结果分析：} 在第三关的设定下，由于天气状态不再是全局确定已知，而是依据先验概率随机出现，
因此我们采用有限时域马尔可夫决策过程与逆向动态规划方法进行求解。
计算结果表明，在该随机环境下，玩家的最大期望最终保留资金为 \textbf{12730 元}。
最优策略表现出明显的“前期推进—中途补给—矿山集中获利—末期冲刺终点”的阶段性特征：
玩家在第 0 天于起点购买 130 箱水和 405 箱食物，并充分利用初始负重上限，为后续长距离推进奠定资源基础；
随后沿 2→10→19→20→28→37→38→39 的路径前进，在前期通过适当停留平衡天气变化带来的资源消耗压力，
并于第 10 天在 39 号村庄补充 211 箱水，以保障后续进入矿山区域的续航能力。
此后玩家经 47→55 到达 55 号矿山，在第 14 天至第 18 天连续挖矿 5 天获取第一阶段收益；
随后于第 19 天前往 62 号村庄补充 174 箱水和 86 箱食物，再于第 20 天返回 55 号矿山，于第 21 天至第 28 天继续挖矿 8 天，
从而在满足终点时限约束的前提下尽可能延长高收益作业时间。
最终玩家于第 29 天经 63 号节点前进，并于第 30 天安全抵达 64 号终点，且水与食物恰好耗尽，最终资金达到 \textbf{12730 元}。
这一结果说明，在随机天气背景下，最优策略并非简单追求最短路径，而是在资源消耗、补给时机、挖矿收益与终点可达性之间实现了全局最优平衡。


与问题一相比，问题二不再输出一条唯一的静态最优路径，而是输出一个\textbf{面向随机天气变化的动态反馈策略}。
该策略并不依赖对未来天气的准确预知，而是根据当天已知天气信息实时调整行动，因此更符合不确定环境下的实际决策逻辑。
模型结果表明，该方法能够在天气随机扰动下兼顾生存性、可达性与盈利性，为玩家提供具有期望最优意义的全过程决策依据。

\subsection{问题三：多玩家条件下的博弈决策模型}

针对问题三，游戏由单玩家扩展为多玩家同时参与。此时，每个玩家的收益不仅取决于自身的路径规划、资源配置与天气演化，还会受到其他玩家行为的显著影响。
由于多人在同一路径移动、同一矿山挖矿或同一村庄购买资源时，都会引发额外消耗、收益平分或价格上涨等拥挤效应，因此该问题本质上是一个多阶段动态博弈问题。

为统一描述问题三两种设定，本文将第 $k$ 个玩家在第 $t$ 天的状态表示为
\[
S_t^{(k)}=(L_t^{(k)},W_t^{(k)},F_t^{(k)},M_t^{(k)},T_t),
\qquad k=1,2,\dots,n,
\]
其中：
\begin{itemize}
    \item $L_t^{(k)}$ 表示第 $k$ 个玩家在第 $t$ 天结束时所在区域；
    \item $W_t^{(k)}$ 表示第 $k$ 个玩家在第 $t$ 天结束时剩余水量；
    \item $F_t^{(k)}$ 表示第 $k$ 个玩家在第 $t$ 天结束时剩余食物量；
    \item $M_t^{(k)}$ 表示第 $k$ 个玩家在第 $t$ 天结束时剩余资金；
    \item $T_t$ 表示第 $t$ 天的天气状态。
\end{itemize}

记所有玩家在第 $t$ 天的联合状态为
\[
\mathbf{S}_t=\bigl(S_t^{(1)},S_t^{(2)},\dots,S_t^{(n)}\bigr).
\]

设第 $k$ 个玩家在第 $t$ 天的决策为
\[
D_t^{(k)}\in\mathcal{A}\bigl(S_t^{(k)}\bigr),
\]
其中
\[
\mathcal{A}\bigl(S_t^{(k)}\bigr)=\{1,2,3\},
\]
并约定
\[
D_t^{(k)}=
\begin{cases}
1, & \text{停留},\\
2, & \text{行走},\\
3, & \text{挖矿}.
\end{cases}
\]

若第 $k$ 个玩家在第 $t$ 天于村庄购买资源，则分别记其购买水和食物数量为
\[
U_t^{(k)},\qquad V_t^{(k)}.
\]

联合决策记为
\[
\mathbf{D}_t=\bigl(D_t^{(1)},D_t^{(2)},\dots,D_t^{(n)}\bigr).
\]

\subsubsection*{1. 资源消耗与拥挤效应刻画}

设晴朗、高温、沙暴三种天气下单位基础耗水量分别为
\[
X_W^1,\ X_W^2,\ X_W^3,
\]
单位基础耗食量分别为
\[
X_F^1,\ X_F^2,\ X_F^3.
\]
则有
\[
X_W(T_t)=X_W^{T_t},\qquad X_F(T_t)=X_F^{T_t}.
\]

在单人情形下，停留、行走、挖矿的基础消耗倍率分别为
\[
\beta(D_t^{(k)})=
\begin{cases}
1, & D_t^{(k)}=1,\\
2, & D_t^{(k)}=2,\\
3, & D_t^{(k)}=3.
\end{cases}
\]

考虑多人拥挤效应后，若第 $t$ 天共有 $m_t^{(ij)}$ 名玩家从区域 $i$ 行走到区域 $j$，则这些玩家中任一人的移动消耗倍率调整为
\[
2m_t^{(ij)}.
\]
因此，对第 $k$ 个玩家而言，其第 $t$ 天的实际耗水量与耗食量分别表示为
\[
C_{t,W}^{(k)}=\beta_t^{(k)} X_W(T_t),
\qquad
C_{t,F}^{(k)}=\beta_t^{(k)} X_F(T_t),
\]
其中 $\beta_t^{(k)}$ 为考虑拥挤后的实际倍率。

若第 $t$ 天共有 $h_t$ 名玩家在同一矿山挖矿，则其中任一人的单日挖矿收益为
\[
\frac{R}{h_t}.
\]

若第 $t$ 天共有 $g_t$ 名玩家在同一村庄购买资源，则每箱资源购买价格调整为
\[
\lambda_t G_W,\qquad \lambda_t G_F,
\]
其中题目规定 $\lambda_t=2$ 或更高倍数时按相应规则修正。为统一起见，记村庄购买价格倍率为 $\lambda_t$。

\subsubsection*{2. 多玩家状态转移方程}

对于第 $k$ 个玩家，资源状态转移满足
\[
W_{t+1}^{(k)}=W_t^{(k)}+U_t^{(k)}-C_{t,W}^{(k)},
\]
\[
F_{t+1}^{(k)}=F_t^{(k)}+V_t^{(k)}-C_{t,F}^{(k)}.
\]

设水和食物的基准价格分别为 $G_W,G_F$，则资金状态转移方程为
\[
M_{t+1}^{(k)}
=
M_t^{(k)}
+Y_t^{(k)}
-\lambda_t G_W U_t^{(k)}
-\lambda_t G_F V_t^{(k)},
\]
其中 $Y_t^{(k)}$ 为第 $k$ 个玩家在第 $t$ 天的挖矿收益，满足
\[
Y_t^{(k)}=
\begin{cases}
\dfrac{R}{h_t}, & D_t^{(k)}=3,\\[6pt]
0, & D_t^{(k)}\neq 3.
\end{cases}
\]

位置状态转移满足
\[
L_{t+1}^{(k)}=
\begin{cases}
L_t^{(k)}, & D_t^{(k)}=1,\\
L_t^{(k)}, & D_t^{(k)}=3,\\
j,\quad j\in\mathcal{N}(L_t^{(k)}), & D_t^{(k)}=2.
\end{cases}
\]

于是，可将系统写为联合状态转移
\[
\mathbf{S}_{t+1}=f(\mathbf{S}_t,\mathbf{D}_t,T_{t+1}).
\]

\subsubsection*{3. 问题三第一小问：天气已知下的静态博弈模型}

当整个游戏时段内天气序列已知时，各玩家需在第 $0$ 天给出完整行动方案，之后不再更改。此时，问题可视为一个完全信息静态博弈。

设第 $k$ 个玩家的终端收益为
\[
H^{(k)}=M_T^{(k)}+\frac12 G_W W_T^{(k)}+\frac12 G_F F_T^{(k)}.
\]

若以系统整体利益最大为目标，则建立如下优化模型：
\[
\max \ \sum_{k=1}^{n} H^{(k)}.
\]

若进一步从博弈均衡角度描述，则设第 $k$ 个玩家的策略为 $\pi^{(k)}$，其收益函数为
\[
J^{(k)}\bigl(\pi^{(1)},\pi^{(2)},\dots,\pi^{(n)}\bigr)=H^{(k)}.
\]
若存在策略组合
\[
\bigl(\pi^{(1)*},\pi^{(2)*},\dots,\pi^{(n)*}\bigr),
\]
使得对任意玩家 $k$ 和任意可行策略 $\pi^{(k)}$，都有
\[
J^{(k)}\bigl(\pi^{(1)*},\dots,\pi^{(k)*},\dots,\pi^{(n)*}\bigr)
\ge
J^{(k)}\bigl(\pi^{(1)*},\dots,\pi^{(k)},\dots,\pi^{(n)*}\bigr),
\]
则称该策略组合构成问题三第一小问的纳什均衡。

\subsubsection*{4. 问题三第二小问：仅知道当天天气下的多玩家随机动态协同决策模型}

对于问题三第二小问，所有玩家仅能在第 $t$ 天开始时观察到当天天气，且在当天行动结束后，
才能获知其他玩家当天的行动方案与剩余资源状态，并据此共同决定第 $t+1$ 天的行动。
因此，该问题适合建模为一个\textbf{带有随机天气扰动和多玩家耦合约束的滚动动态决策问题}。

结合题目规则可知，多名玩家之间并非完全独立：若若干玩家在同一天沿同一路径移动，
则行走资源消耗将提升；若在同一矿山同时挖矿，则单位玩家收益下降；若在同一村庄同时购买，
则补给成本上升。故每位玩家在第 $t$ 天的最优动作不仅依赖于自身状态，也依赖于其他玩家的当日状态与协同行为。:contentReference[oaicite:1]{index=1}

\paragraph{（1）系统状态的定义}

设共有 $n$ 名玩家，第 $k$ 名玩家在第 $t$ 天结束后的状态定义为
\[
s_t^{(k)}=\bigl(i_t^{(k)},\,W_t^{(k)},\,F_t^{(k)},\,M_t^{(k)}\bigr),
\]
其中：
\begin{itemize}
    \item $i_t^{(k)}$ 表示玩家 $k$ 在第 $t$ 天结束时所在节点；
    \item $W_t^{(k)},F_t^{(k)}$ 分别表示玩家 $k$ 剩余水量与食物量；
    \item $M_t^{(k)}$ 表示玩家 $k$ 的剩余资金。
\end{itemize}

于是第 $t$ 天的系统联合状态可表示为
\[
S_t=\Bigl(t,\omega_t,s_t^{(1)},s_t^{(2)},\ldots,s_t^{(n)}\Bigr),
\]
其中 $\omega_t\in\{\text{晴朗},\text{高温},\text{沙暴}\}$ 表示第 $t$ 天天气。

\paragraph{（2）动作集合的定义}

设第 $k$ 名玩家在第 $t$ 天采取的动作记为
\[
a_t^{(k)}\in\mathcal{A}_t^{(k)}=\{\text{停留},\text{移动},\text{挖矿},\text{购买}\}.
\]
则全体玩家在第 $t$ 天的联合动作可写为
\[
A_t=\bigl(a_t^{(1)},a_t^{(2)},\ldots,a_t^{(n)}\bigr).
\]

其中各动作需满足以下规则约束：
\begin{itemize}
    \item 若 $\omega_t=\text{沙暴}$，则所有玩家均不得执行移动动作；
    \item 仅当玩家位于矿山且不属于“到达矿山当天”时，方可执行挖矿动作；
    \item 仅当玩家位于村庄时，方可执行购买动作；
    \item 任意玩家均需满足资源非负约束、负重约束与截止日期约束。
\end{itemize}

\paragraph{（3）多人耦合规则的刻画}

记在第 $t$ 天从节点 $u$ 移动到节点 $v$ 的玩家集合为
\[
\mathcal{G}_{uv}(t)=\{k:\ i_{t-1}^{(k)}=u,\ i_t^{(k)}=v,\ u\neq v\},
\]
其人数记为
\[
g_{uv}(t)=|\mathcal{G}_{uv}(t)|.
\]
若 $g_{uv}(t)\ge 1$，则集合中任一玩家的行走消耗倍率由单人情况下的 $2$ 倍变为
\[
2g_{uv}(t).
\]

类似地，记第 $t$ 天在同一矿山挖矿的玩家数为 $h_m(t)$，则该矿山中每名玩家的挖矿收益为
\[
\frac{R_0}{h_m(t)},
\]
其中 $R_0$ 为基础挖矿收益。

记第 $t$ 天在同一村庄购买资源的玩家数为 $q_v(t)$，则该村庄内每箱资源的购买价格提升为
\[
\lambda\bigl(q_v(t)\bigr)p,
\]
其中 $p$ 为基准价格，$\lambda(\cdot)$ 为随购买人数变化的价格倍率函数。

\paragraph{（4）状态转移关系}

在给定联合状态 $S_t$ 与联合动作 $A_t$ 后，第 $t+1$ 天的系统状态由天气实现值与资源结算共同决定：
\[
S_{t+1}=\mathcal{T}(S_t,A_t,\omega_{t+1}),
\]
其中 $\mathcal{T}(\cdot)$ 表示状态转移函数。对任一玩家 $k$，其资源与资金更新满足
\[
W_{t+1}^{(k)}=W_t^{(k)}-\Delta W_t^{(k)}+B_{W,t}^{(k)},
\]
\[
F_{t+1}^{(k)}=F_t^{(k)}-\Delta F_t^{(k)}+B_{F,t}^{(k)},
\]
\[
M_{t+1}^{(k)}=M_t^{(k)}+R_t^{(k)}-C_t^{(k)},
\]
其中：
\begin{itemize}
    \item $\Delta W_t^{(k)},\Delta F_t^{(k)}$ 分别表示玩家 $k$ 在第 $t$ 天消耗的水和食物；
    \item $B_{W,t}^{(k)},B_{F,t}^{(k)}$ 分别表示当日在村庄补给的水和食物；
    \item $R_t^{(k)}$ 表示挖矿收益；
    \item $C_t^{(k)}$ 表示购买资源的花费。
\end{itemize}

\paragraph{（5）优化目标}

由于第六关未来天气未知，且联合状态空间维数极高，本文不直接追求解析意义下的全局均衡解，
而是设计\textbf{状态反馈型协同策略} $\Pi=\{\pi^{(1)},\pi^{(2)},\ldots,\pi^{(n)}\}$，
使玩家在每一天根据当前可观测信息滚动调整动作。为兼顾收益性与生存性，定义团队层面的优化目标为
\[
\max_{\Pi}\ \mathbb{E}\!\left[\sum_{k=1}^{n} M_T^{(k),\mathrm{end}}\right],
\]
其中 $M_T^{(k),\mathrm{end}}$ 表示第 $k$ 名玩家在终点清算剩余物资后的最终资金。

同时，为保证方案具有较强鲁棒性，还要求策略尽可能提高全员存活概率与全员到达概率，即
\[
P\bigl(N_{\mathrm{alive}}=n\bigr)\ \text{尽可能大}, \qquad
P\bigl(N_{\mathrm{arrive}}=n\bigr)\ \text{尽可能大}.
\]

因此，问题三第二小问可概括为：在随机天气背景下，构造满足多人耦合规则的滚动反馈协同策略，使团队总收益保持较高水平，并尽量提高全员存活率与终点可达率。

\paragraph{（6）求解思想}

考虑到该问题同时包含随机天气、多玩家耦合和逐日信息更新三重复杂性，
若直接对联合状态空间进行精确动态规划，将导致维数爆炸，计算上不可行。
因此，本文采用如下两阶段求解思路：

\begin{itemize}
    \item \textbf{策略构造阶段：} 基于当前位置、剩余资源、天气与其他玩家状态，
    设计“错峰移动、分散挖矿、分时补给、优先保障生存”的滚动反馈协同规则；
    \item \textbf{策略评估阶段：} 在给定天气概率分布下，采用蒙特卡洛方法重复生成随机天气序列，
    对所构造策略进行大量仿真，从而估计其平均收益、波动水平、存活概率与到达概率。
\end{itemize}

由此，第六关可归纳为\textbf{基于状态反馈的多玩家随机动态协同决策问题}。

\subsubsection*{6. 问题三统一模型表示}

综上，问题三可统一表示为如下多玩家动态博弈模型：
\[
\max \ \sum_{k=1}^{n}\mathbb{E}\bigl(H^{(k)}\bigr)
\]

\[
\mathrm{s.t.}\quad
\left\{
\begin{aligned}
\mathbf{S}_t      & = \bigl(S_t^{(1)},S_t^{(2)},\dots,S_t^{(n)}\bigr),\\
S_t^{(k)}          & = (L_t^{(k)},W_t^{(k)},F_t^{(k)},M_t^{(k)},T_t),\\
W_{t+1}^{(k)}      & = W_t^{(k)}+U_t^{(k)}-C_{t,W}^{(k)},\\
F_{t+1}^{(k)}      & = F_t^{(k)}+V_t^{(k)}-C_{t,F}^{(k)},\\
M_{t+1}^{(k)}      & = M_t^{(k)}+Y_t^{(k)}-\lambda_tG_WU_t^{(k)}-\lambda_tG_FV_t^{(k)},\\
L_{t+1}^{(k)}      & \in \mathcal{N}(L_t^{(k)})\cup\{L_t^{(k)}\},\\
C_{t,W}^{(k)}      & = \beta_t^{(k)}X_W(T_t),\\
C_{t,F}^{(k)}      & = \beta_t^{(k)}X_F(T_t),\\
Y_t^{(k)}          & =
\begin{cases}
\dfrac{R}{h_t}, & D_t^{(k)}=3,\\
0, & D_t^{(k)}\neq 3,
\end{cases}\\
H^{(k)}            & = M_T^{(k)}+\dfrac12 G_WW_T^{(k)}+\dfrac12 G_FF_T^{(k)},\\
                   & 3W_t^{(k)}+2F_t^{(k)} \le H_{\max},\\
                   & W_t^{(k)},F_t^{(k)},M_t^{(k)} \ge 0,\\
                   & T_t \in \{1,2,3\},\\
                   & D_t^{(k)} \in \{1,2,3\},\\
                   & T_t=3 \Rightarrow D_t^{(k)}\neq 2,\\
                   & D_t^{(k)}=3 \Rightarrow L_t^{(k)}\in\mathcal{M},\ L_{t-1}^{(k)}=L_t^{(k)},\\
                   & U_t^{(k)}>0\ \text{or}\ V_t^{(k)}>0 \Rightarrow L_t^{(k)}\in\mathcal{V},\\
L_0^{(k)}          & = L_{\mathrm{start}},\quad L_T^{(k)}=L_{\mathrm{end}},\\
t                  & \le T_{\max}.
\end{aligned}
\right.
\]

由此，问题三可分别从完全信息静态博弈与不完全信息动态博弈两个层面进行描述：前者强调全局行动方案的一次性确定，后者强调在每日新信息到来后进行滚动优化决策。

\subsubsection*{5. 第六关蒙特卡洛模拟结果分析}

为评估上述多玩家随机动态协同策略在随机天气环境下的稳定性与有效性，
本文在给定天气概率分布下进行了大量蒙特卡洛重复试验，
统计得到最终总资金、存活人数、到达终点人数及关键概率指标。

模拟结果如下：
\[
\text{平均最终总资金}=23945.62\ \text{元},\qquad
\text{标准差}=2933.47\ \text{元},
\]
\[
\text{平均存活人数}=2.919,\qquad
\text{平均到达终点人数}=2.919,
\]
\[
P(\text{全员存活})=92.90\%,\qquad
P(\text{全员到达})=92.90\%,\qquad
P(\text{至少1人到达})=100.00\%.
\]

此外，各玩家终端资金均值与标准差分别为
\[
\text{玩家0： } 9316.34 \pm 1090.51,
\]
\[
\text{玩家1： } 8272.18 \pm 1431.02,
\]
\[
\text{玩家2： } 6357.10 \pm 1690.94.
\]

天气样本频率估计为
\[
P(\text{晴朗})=50.53\%,\qquad
P(\text{高温})=39.51\%,\qquad
P(\text{沙暴})=9.96\%.
\]

基于上述统计结果，可得到以下分析结论。

\paragraph{（1）生存与到达指标}
全员存活概率与全员到达概率均为 $92.90\%$，且至少 1 人到达终点的概率为 $100\%$。
同时，平均存活人数与平均到达人数均为 $2.919$。这表明策略在多数样本路径下能够保证任务完成，主要风险集中在少量极端天气样本。

\paragraph{（2）团队收益水平}
平均最终总资金为 $23945.62$ 元，标准差为 $2933.47$ 元。结合分布图可见，收益样本主要集中在中高区间，说明策略在保证较高生存率的同时能够维持较稳定的收益水平。

\paragraph{（3）个体收益差异}
三名玩家的终端资金均值存在差异，且玩家2的波动相对更大，说明当前协同策略在收益分配上更偏向团队目标，而非个体收益均衡。

\paragraph{（4）天气样本一致性}
晴朗、高温、沙暴的经验频率分别为 $50.53\%$、$39.51\%$、$9.96\%$，与设定分布基本一致，说明天气采样机制具有一致性，相关统计结果具有可解释性。

综合来看，第六关策略在生存率、到达率与团队收益之间实现了较好的平衡，但在极端天气场景下仍存在收益下降风险。

\section{模型评价与改进}

\subsection{模型的优点}

\begin{itemize}
    \item \textbf{统一建模结构清晰：} 各关卡在同一“状态—决策—收益”框架下展开，符号体系一致，便于比较不同信息结构下的决策差异。
    \item \textbf{确定性场景可获得精确解：} 对于天气已知情形，采用MILP能够在约束体系内给出可验证的最优解，结果可复现实验过程。
    \item \textbf{随机场景求解效率较高：} 对于问题二，采用逆向动态规划并结合状态简化策略，能够在可接受时间内完成价值函数计算。
    \item \textbf{多玩家策略可评估性较好：} 对于问题三，滚动决策与蒙特卡洛评估相结合，可同时给出收益指标和生存指标，便于进行综合比较。
\end{itemize}

\subsection{模型的缺点}

\begin{itemize}
    \item \textbf{对先验分布依赖较强：} 随机场景中策略质量依赖天气概率设定。当真实分布与先验偏离时，期望最优策略可能出现性能下降。
    \item \textbf{多玩家部分属于近似求解：} 第六关采用滚动评估与局部更新，具有较好的可计算性，但未直接给出全局均衡意义下的精确解。
    \item \textbf{行为假设理想化：} 当前模型默认玩家以收益最大化为目标，未显式刻画有限理性与风险偏好差异，可能导致与真实行为存在偏差。
\end{itemize}

\subsection{模型的改进}

\begin{itemize}
    \item \textbf{引入鲁棒目标函数：} 在期望收益目标之外，可加入CVaR等风险指标，提升策略在极端天气样本下的稳健性。
    \item \textbf{动态更新天气分布：} 可结合贝叶斯更新或隐马尔可夫模型，根据历史观测调整天气概率，提高策略的环境适应性。
    \item \textbf{完善多玩家求解方法：} 后续可进一步尝试多智能体强化学习或更高阶近似动态规划，以提升高维耦合场景下的求解质量。
\end{itemize}
